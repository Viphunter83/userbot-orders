# Проверка установки проекта

## Быстрая проверка

1. **Установка зависимостей:**
```bash
pip install -r requirements.txt
```

2. **Проверка модуля настроек:**
```bash
python -m src.config.settings
```
Ожидается ошибка о недостающих переменных окружения (это нормально, если .env не заполнен)

3. **Проверка модуля логирования:**
```bash
python -m src.utils.logger
```
Должно вывести тестовые сообщения логов

4. **Запуск тестов:**
```bash
pytest
```
Тесты должны пройти (могут быть предупреждения, если .env не настроен)

## Структура проекта

```
userbot-orders/
├── src/
│   ├── config/          ✓ settings.py
│   ├── utils/           ✓ logger.py
│   └── [другие модули]  ✓ __init__.py файлы
├── tests/               ✓ test_settings.py, test_logger.py
├── .env.example         ✓ шаблон конфигурации
├── .env                 ✓ (защищен gitignore)
├── requirements.txt     ✓ зависимости
├── pytest.ini          ✓ конфигурация тестов
└── README.md            ✓ документация
```

## Следующие шаги

1. Заполните `.env` файл реальными значениями
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите проверку: `pytest`
